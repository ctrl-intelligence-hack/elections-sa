Elections South Africa

An easy way to find, explore and reuse South African's election data

Use the free API

The Elections South Africa API is simple to use: we've designed HTTP GET requests to retrieve past election results, upcoming elections, elected govenment officals and elections current news in our database (in JSON format), so you can see everything we're tracking and update your own records as regularly as you need. The Elections South Africa API includes, General/National elections Provincial and Municipal elections.

You can use this API to design your own reminders, visualizations, outreach campaigns, and much more.
